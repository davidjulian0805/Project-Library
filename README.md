# 📚 Library App

A simple **Library application** built with **HTML, CSS, and JavaScript** as part of **The Odin Project JavaScript curriculum**.
This project allows users to add books, display them in the library, toggle their read status, and remove them from the collection.

## 📖 Features

* Add new books using a form
* Display books in a card layout
* Remove books from the library
* Toggle read/unread status
* Each book has a **unique ID** using `crypto.randomUUID()`
* Data is stored in a **JavaScript array**
* Dynamic DOM rendering of books
